Тест-файлы расположены в `data/` (не в репозитории).

| # | Сценарий | Вход | Ожидаемый результат | Фактический результат | Статус |

| 3 | **Загрузка нескольких файлов** | БДР + ДДС одновременно | Оба файла приняты; нормализованный контекст содержит `pnl` и `cashflow` | | |
| 4 | **Определение периода** | Файл с заголовком «Январь–Март 2026» | `period.period_label` = «за 1 квартал 2026 года» | | |

| 10 | **Fallback без ключа OpenAI** | Запрос Q&A при недействительном `OPENAI_API_KEY` | Сообщение об ошибке вместо краша; парсинг и нормализация продолжают работать | | |

## Инструкция по заполнению

1. Запустить приложение: `uvicorn app.main:app --reload --port 8010`
2. Пройти каждый сценарий вручную или через тесты в `tests/`
3. Заполнить колонки «Фактический результат» и «Статус» (Пройден / Провален / Пропущен)

## Примечания

- Сценарии 7–9 требуют действительного `OPENAI_API_KEY` в `.env`
- Сценарий 10 специально тестируется при отсутствующем или недействительном ключе
- Числовые значения в сценарии 6 взяты из реального тестового отчёта ЗМГ Q1 2026
- Для автоматической проверки сценариев 8–9 см. `tests/smoke_test_n4.py` и `tests/smoke_test_n5.py`
