# Расширенный отчет для собственника

Расширенный отчет используется, если загружено несколько файлов, есть БДР и ДДС или требуется подробный управленческий вывод.

Требования:
- минимум 5 таблиц;
- после каждой таблицы один короткий вывод;
- все ключевые цифры указывать с источником: БДР или ДДС;
- каждый блок начинается с главного смысла;
- нельзя писать длинные теоретические пояснения.

Обязательные таблицы:
1. Экономика бизнеса / БДР
2. Структура расходов
3. Денежный поток / ДДС
4. Прибыль vs деньги
5. KPI собственника
