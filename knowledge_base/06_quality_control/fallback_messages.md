# Fallback-сообщения

Если файл не распознан:
"Файл загружен, но данные не удалось надежно извлечь. Попробуйте загрузить Excel или PDF с текстовым слоем."

Если нет БДР:
"БДР не найден. Выводы по прибыли и марже будут ограничены."

Если нет ДДС:
"ДДС не найден. Выводы по деньгам и кассовому разрыву будут ограничены."

Если произошел сбой API:
"Не удалось сформировать AI-отчет. Данные обработаны, попробуйте повторить формирование отчета позже."
