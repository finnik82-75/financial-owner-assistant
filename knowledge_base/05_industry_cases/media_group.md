# Медиагруппа

Для медиагруппы важно анализировать направления отдельно.

Одни направления могут быть донорами, другие поглотителями.
Прибыльные направления могут субсидировать убыточные.

Частые зоны риска:
- высокий ФОТ;
- слабые проекты с отрицательной маржей;
- субсидирование слабых направлений;
- операционный денежный минус;
- зависимость от внутригруппового финансирования.

# Проектный разрез для медиагруппы

Для медиагруппы проектный разрез обязателен.

Типовые проекты:
- Европа+
- Авторадио
- Ретро FM
- Сайт "Забmedia.ru"
- ЗАБ ТВ 24
- Наружная реклама

В отчете нужно отдельно показывать:
- доноров прибыли — проекты с положительной операционной прибылью;
- поглотителей прибыли — проекты с отрицательной операционной прибылью;
- проекты с отрицательной операционной прибылью по каждому месяцу;
- проекты, которые ухудшили результат в конкретном месяце.

Ключевые вопросы проектного анализа:
- какой проект принес наибольшую выручку за период;
- какой проект создал наибольший убыток;
- есть ли проект, убыток которого покрывается за счет других;
- в каком месяце убыток по проекту был максимальным.
