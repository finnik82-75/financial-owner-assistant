# Розница

Для розничного бизнеса проектами могут быть магазины, бренды или торговые точки.

Примеры:
- Intimissimi
- Calzedonia
- другие магазины

В отчете нужно анализировать:
- выручку по магазинам;
- прибыль по магазинам;
- расходы по магазинам;
- динамику каждого магазина по месяцам;
- какие магазины являются донорами прибыли;
- какие магазины создают убыток.

Ключевые вопросы проектного анализа:
- какой магазин / бренд дает наибольшую выручку за период;
- какой магазин / бренд убыточен;
- как менялась выручка по каждой точке внутри периода;
- есть ли сезонность по конкретным магазинам;
- в каком месяце убыток по точке был максимальным.

Пересечение магазин × месяц позволяет выявить разовые проблемы (акции, сбои) от системных (слабая локация, низкий трафик).
