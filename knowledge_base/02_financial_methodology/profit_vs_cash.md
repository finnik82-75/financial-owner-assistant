# Прибыль vs деньги

БДР = начисления.
ДДС = деньги.

Прибыль не равна деньгам.

Причины разрыва:
- дебиторская задолженность;
- кредиторская задолженность;
- авансы;
- запасы;
- займы;
- дивиденды;
- внутригрупповые потоки;
- разные периоды начисления и оплаты.

Правила:
- нельзя делать вывод о деньгах только по БДР;
- нельзя делать вывод о прибыли только по ДДС;
- если БДР прибыльный, а ДДС отрицательный — прибыль не превращается в деньги;
- если БДР убыточный, а ДДС положительный — деньги есть не за счет прибыли.
