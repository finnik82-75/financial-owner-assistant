# БДР

БДР показывает экономику по начислению.

Структура:
- Выручка
- Себестоимость
- Валовая прибыль
- Операционные расходы
- ФОТ
- Аренда
- Налоги
- ВГО
- Прибыль от основной деятельности
- Прочие доходы и расходы
- Чистая прибыль

Если чистая прибыль отсутствует, допускается использовать proxy:
- прибыль от основной деятельности;
- операционная прибыль.

Proxy нужно явно отмечать в отчете.

# Три измерения анализа БДР

БДР нужно анализировать в трех измерениях:

**1. Итог за отчетный период**
Суммарные значения за период (квартал, полугодие, год).
Отвечает на вопрос: каков итоговый результат.

**2. Помесячная динамика**
Значения по каждому месяцу внутри периода.
Отвечает на вопрос: как менялся результат внутри периода.

**3. Проектный разрез**
Значения по каждому проекту или направлению бизнеса.
Отвечает на вопрос: какой проект дает результат, а какой создает убыток.

# Проектный разрез БДР

Если в отчете есть проекты или направления, БДР нужно анализировать в трех разрезах:
- общий итог по компании;
- помесячная динамика;
- проектный разрез.

Проектный анализ должен отвечать на вопросы:
- какой проект дает выручку;
- какой проект дает прибыль;
- какой проект создает убыток;
- какие прибыльные проекты субсидируют убыточные;
- какой проект ухудшил результат внутри конкретного месяца.

Пересечение проект × месяц:
Если данные позволяют, нужно показать результат каждого проекта в каждом месяце.
Это позволяет выявить, была ли проблема разовой (один месяц) или системной (все месяцы).
