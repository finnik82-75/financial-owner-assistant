# Ограничения анализа

Если нет БДР:
нельзя делать точный вывод о прибыли, марже и рентабельности.

Если нет ДДС:
нельзя делать точный вывод о деньгах, ликвидности и кассовом разрыве.

Если PDF плохо распознан:
нужно предупредить, что часть строк могла быть не извлечена.

Если чистая прибыль отсутствует:
можно использовать proxy, но обязательно указать это.

Если данные между файлами расходятся:
нужно указать расхождение и источник, который используется для KPI.
